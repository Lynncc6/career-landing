# Landing 清单（前 3 周 / 前 3 月 · AI Agent 产品运营版）

> 测试案例 persona：小柒，校招应届生，AI Agent 公司产品运营（类似 WorkBuddy），入职第 1 周。
> 目标：从"小白打杂感"变成"懂用户、懂产品边界的运营"。

## 前 3 周
- [ ] 找 leader 对齐前 3 月最看重的 3 件事（拉新/活跃/留存/反馈闭环？）
- [ ] 画出人际地图（leader/带教/算法接口人/产品/研发）
- [ ] 跑通产品后台 + 数据看板 + 用户反馈渠道 + A/B 实验入口
- [ ] 读懂 agent 能力边界文档，能用大白话讲清"我们的 agent 能干嘛、不能干吗"
- [ ] 搭好 AI 运营基础设施：提示词库 + 用户反馈整理模板 + 周报模板
- [ ] 用 AI 把用户反馈分类 / 写周报草稿，省出时间搞明白
- [ ] 本周问出一个"傻问题"（如"这个指标为啥这么算"）并记录没人笑你

## 前 3 月 · 第 1 月
- [ ] 独立发一次用户调研 / 梳理一份反馈分类 / 跑一个小 A/B
- [ ] 交出的东西自己先检查（数据对不对、结论站不站得住）
- [ ] 发出第一份主动进度同步（让 leader 知道你卡在哪）

## 前 3 月 · 第 2 月
- [ ] 接活前先确认"好"的标准（看活跃 / 留存 / 反馈闭环）
- [ ] 做一件超出预期的小事（把 user feedback 整理成算法/产品能用的结构化输入模板）
- [ ] 验证一次 AI 给的运营结论，记进翻车记录

## 前 3 月 · 第 3 月
- [ ] 认领一个运营模块（某垂类 / 某渠道 / 某反馈闭环）
- [ ] 写一份自己的方法论备忘（用户反馈→产品的输入规范）
- [ ] 做一次"脱离 AI 裸做"自检（关掉 AI 手写一份用户反馈洞察）
- [ ] 认真想一次方向：深耕产品运营 / 往 AI 产品靠 / 往数据分析靠？

## 每周复盘
- 这周做了：__
- 卡在哪（数据 / 产品边界 / 跨团队 / 自己）：__
- 下周重点：__
- 一个 AI 帮到我的点 / 一个 AI 坑了我的点：__
- 这周有没有靠近过一个真实用户 / 读一条原始反馈：__
