# Landing 框架：前 3 周 / 前 3 月

两个大阶段，前 3 月细分成第 1 / 2 / 3 月三个小节点。每阶段给：目标 / 关键动作 / 检查点 / 危险信号。调性：人话、敢给判断。

> 核心前提：AI 时代没有公司给你安排好的"安全缓冲期"了，landing 期得你自己抢、自己设计。这份框架就是帮你把那段缓冲抢回来。

---

## 阶段一：前 3 周 —— 活下来，搞明白

**目标**：别慌，先把"我是谁、该找谁、要交什么"这三件事弄明白。前两周不出成绩没人怪你，但"听不懂、问不出、找不到人"会让人觉得你不行。

**关键动作**
- 搞清预期：主动找 leader 对齐"你这段时间最看重我做成哪 3 件事"——是**研究导向**（出指标/论文）还是**业务导向**（上线功能/跑通 demo）？记下来，别等别人安排。
- 画人际地图：认识 leader、带教/mentor、同组研究员/工程师、跨组接口人（数据 owner / 平台 / 产品）。先知道"算力找谁申请、数据找谁要、卡住问谁"。
- 摸环境：把实验平台、算力申请、数据权限、代码仓库、评测（Eval）流程跑通一遍。别每次卡在"这脚本在哪、这卡怎么申请"。
- 用 AI 搭基础设施：让 AI 帮你解释组内核心论文、整理术语表、建提示词库和实验记录模板。省出时间"搞明白"。

**检查点（第 3 周末自问）**
- 我能说清 leader 对我前 3 月的核心期待（研究导向还是业务导向）吗？
- 遇到实验/数据/算力问题，我知道该找谁、走什么流程吗？
- 我有没有读过组内一篇核心论文、或跑通过一次 baseline？

**危险信号**
- 三周了还不敢问"这个参数为啥这么设"这种傻问题（氛围或心态得调）
- 每天很忙但说不清在忙啥（没有预期锚点）
- 完全没碰过实验平台 / 没申请到算力（在 AI 公司这是硬隐患，不是省心）

## 阶段二：前 3 月 —— 立住，出成绩

**目标**：从"能干活"变成"这人靠谱、能扛事"。前 3 月是试用期生死线，大部分"不合适"的判断都发生在这。

### 第 1 月：独立跑通一个实验 / 交付一个功能，立"靠谱"人设
- 别急着表现聪明，先把"答应的事准时交、交之前自己检查过"这一个人设立住——**靠谱比聪明值钱得多**。
- 算法/研究：独立 repro 一个 baseline，把指标跑出来（哪怕没涨点，跑通就赢）。
- 数据/ML 工程：独立跑通一条训练/推理管线，交付一个可用模块。
- AI 产品/运营/售前：交付一个能 demo 的功能 / 一次增长实验 / 一份客户场景库。
- 主动同步进度：周报让 leader 知道你卡在算力、数据还是设计，别闷头耗。
- 用 AI 放大产出，但关键判断自己留痕（见 ai-era-module）。

### 第 2 月：对齐"模型效果 vs 上线"的取舍，用 AI 但不被带偏
- 接到活先确认"做成什么样算好"——是抠指标还是保上线，少返工。
- 研究岗：开始做 ablation，理解每个改动为什么涨/不涨；工程岗：把脚本改成自己设计的流程，关注稳定性。
- 产品/售前：把"技术能做什么"翻译成"用户/客户要什么"。
- 做一件"超出预期一点点"的事：比如补齐实验记录模板、写一份新人 onboarding 文档、整理常见坑。
- 验证 AI 的输出，建立自己的判断标准（见 ai-era-module）。

### 第 3 月：承担稍大模块，提改进，沉淀方法论，想方向
- 主动认领一个稍大的任务或一个小改进，让人看到"你能想事"。
- 把前 3 月踩的坑、攒的套路写成一份属于自己的备忘/方法论（实验记录规范 / 管线 checklist）。
- 复盘方向：做研究、做业务、还是转产品/售前？（给"方向迷茫"和"研究 vs 落地撕扯"一个真实出口）

**检查点（第 3 月末自问）**
- leader 愿意把"重要但不紧急"的活/模块交给我吗？
- 我有 1 件"这是我做的、有我印记"的事吗（一篇 repro 笔记 / 一个上线模块）？
- 脱离 AI 我还能不能把核心活干下来（手写一个推导、裸跑一个脚本）？防脆皮
- 我对"做研究 vs 做业务 vs 转方向"有模糊但真实的判断了吗？

**危险信号**
- 三个月还在只跑别人给的脚本、从没自己设计过实验（可能被定型为"跑数工具人"）
- leader 对你的评价还停在"态度不错"（没看到技术判断）
- 从没读过组内核心论文 / 讲不清自己模块的原理（地基虚）
- 全程没独立思考过方向（试用期过了也迷茫）

---

## 贯穿四件事（不管哪个阶段都得做）
1. **预期管理**：定期对齐"什么是好"，比埋头苦干重要。
2. **向上管理**：主动同步 + 敢要资源 + 会汇报，不是拍马屁，是降低信息差。
3. **人际**：1:1 吃饭、请教学长姐，关系是最便宜的信息源。
4. **补地基**：每天留一点时间补底层能力，别全靠 AI 顶着。
